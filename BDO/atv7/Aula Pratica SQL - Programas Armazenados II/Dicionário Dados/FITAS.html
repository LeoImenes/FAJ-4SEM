Table FITAS

Tabela para amazenas informações sobre as diversas cópias fisicas de um Filme.
Columns
Name Type Optional Default Comments
COD_FITA NUMBER     Chave primária da Tabela, formada por um número sequencial.
DESCRICAO VARCHAR2(60)     Descrição da Cópia, para fins administrativos apenas.
DATA_COMPRA DATE     Data em que a cópia foi adquirida pela locadora.
COD_FILME NUMBER     Chave estrangeira da tabela FILME e indica a qual fime a copia se relaciona.
DISPONIVEL VARCHAR2(1)     FLAG para indicar se uma cópia está disponível para Locação ou não. Os valores possíveis são S, indicando cópia disponível ou N, indicando que a cópia está alugada no momento.
