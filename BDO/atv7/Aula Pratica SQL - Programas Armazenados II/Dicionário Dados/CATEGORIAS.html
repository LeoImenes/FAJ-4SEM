Table CATEGORIAS

Tabela para armazenar as categorias dos Filmes existentes na Locadora. Basicamente serve para compor o preço da locação de um filme.
Columns
Name Type Optional Default Comments
COD_CATEGORIA NUMBER     Chave primária da tabela, formada por um número sequencial.
DESCRICAO VARCHAR2(80)     Descrição da Categoria. É o nome exibido para os operadores do Sistema.
PRECO_LOCACAO NUMBER(5,2)     Preço cobrado pela locação de um determinado Filme pertencente à Categoria
