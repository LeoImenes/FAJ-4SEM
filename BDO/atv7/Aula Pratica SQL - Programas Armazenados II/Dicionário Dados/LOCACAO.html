Table LOCACAO

Tabela para armazenar informações sobre as locações efetuadas pela locadora. Pode conter tanto as locações passadas quando as locações correntes (ainda não devolvidas).
Columns
Name Type Optional Default Comments
COD_CLIENTE NUMBER     Chave estrangeira da tabela CLIENTE e indica qual o cliente que alugou a cópia de um Filme.
COD_FITA NUMBER     Chave estrangeira da tabela FITA e indica qual cópia do Filme foi locada.
DATA_LOCACAO DATE     Data em que a locação foi feita.
DATA_DEVOLUCAO DATE Y   Data de devolução da locação. Só deverá ser preenchida no momento da devolução da fita. Quando possuir valor NULL, indica que o cliente ainda está com a cópia. Quando valores NOT NULL, indica a data em que a cópia foi devlvida.
VALOR_LOCACAO NUMBER(5,2)     Valor do aluguel.
