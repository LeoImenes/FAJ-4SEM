Table DISCIPLINA

Tabela para armazenar as informações sobre as diversas disciplinas existentes em uma instituição de ensino.
Columns
Name Type Optional Default Comments
CODIGO NUMBER     Chave primária da tabela, formada por um número sequencial.
NOME VARCHAR2(50)     Nome da Disciplina.
DIA VARCHAR2(15)     Dia da semana em que a disciplina é ofereciada. Pode conter valores tais como segunda-feira, terça-feira, quarta-feira, etc.
SALA NUMBER     Número da sala onde é ministrada a disciplina.
PROFESSOR NUMBER     Chave estrangeira da tabela PROFESSOR e indica qual é o professor responsável por ministrar a disciplina.
