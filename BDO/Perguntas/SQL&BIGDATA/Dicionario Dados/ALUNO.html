Table ALUNO

Tabela para armazenar as informações sobre os alunos de uma instituição de ensino.
Columns
Name Type Optional Default Comments
RA NUMBER     Chave primária da tabela, com o número do Registro Acadêmico do aluno.
NOME VARCHAR2(200)     Nome Completo do Aluno.
ORIENTADOR NUMBER Y   Chave estrangeira da tabela PROFESSOR e identifica qual é o professor orientador do aluno. Quando NULL, significa que o aluno não possui orientador.
CODIGO_CIDADE NUMBER     Chave estangeira da tabela CIDADE e identifica qual a cidade onde o aluno reside.
