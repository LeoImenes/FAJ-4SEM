Table FORNECEDOR

Tabela para armazenar os Fornecedores que terão suas entregas controladas pelo Sistema de Controle de Vida Útil dasFerramentas.
Columns
Name Type Optional Default Comments
S_NOME_FANTASIA VARCHAR2(120)     Nome Fantasia do Fornecedor.
ID_FORNECEDOR NUMBER     Chave primaria da tabela, formado por um número sequencial.
S_ATIVO VARCHAR2(1)   'S' FLAG para indicar se o Fornecedor está Ativo (S) ou Inativo (N). Apenas fornecedores cadastrados como ativos podem ser usados para criação de novas informações. Por padrão assume o valor (S).
S_CNPJ VARCHAR2(18)     Número do CNPJ da Empresa, conforme cadastro no SAP.
S_RAZAO_SOCIAL VARCHAR2(20)     Razão Social do Fornecedor.
S_ENDERECO VARCHAR2(200)     Endereço completo do Fornecedor.

XPKFORNECEDORES ID_FORNECEDOR
Check Constraints
Name Condition
CK_FORNECEDOR_ATIVO S_ATIVO IN ('S', 'N')
