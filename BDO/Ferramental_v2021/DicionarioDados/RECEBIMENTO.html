Table RECEBIMENTO

Tabela para armazenar os recebimentos de Peças produzidas por um fornecedor usando um determinada ferramenta.
Columns
Name Type Optional Default Comments
ID_RECEBIMENTO NUMBER     Identificador do Recebimento. Número sequencial gerado automaticamente pelo Sistema.
D_DATA_RECEBIMENTO DATE     Data do Recebimento.
N_QUANTIDADE_RECEBIDA NUMBER     Quantidade recebida.
S_NOTA_FISCAL VARCHAR2(10) Y   Número da Nota Fiscal. Essa informação não será preenchida quando o recebimento for referente a um Scrap.
ID_FORNECEDOR NUMBER     Chave estrangeira da Tabela FORNECEDOR e identifica qual ó Fornecedor envolvido no recebimento.
ID_ALOCACAO_FERRAMENTA NUMBER     Chave estrangeira da tabela ALOCACAO_FERRAMENTA e indica para qual alocação de Ferramenta deverá ser contabilizado o recebimento.
ID_PECA NUMBER     Chave estangeira da tabela PECA e indica qual a Peça que está sendo recebida.
