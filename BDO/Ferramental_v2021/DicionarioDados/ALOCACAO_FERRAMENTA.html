Table ALOCACAO_FERRAMENTA

Tabela contendo a Alocação de uma ferramenta em um determinado fornecedor produzindo determinada peça.
Columns
Name Type Optional Default Comments
ID_FORNECEDOR NUMBER     Chave estangeira da tabela FORNECEDOR e identifica qual é o Fornecedor que está usando a Ferramenta alocada para produzir a peça especificada.
ID_FERRAMENTA NUMBER     Chave estrangeira da Tabela FERRAMENTA e identifica qual é a Ferramenta alocada ao Fornecedor para produzir determinada Peça.
ID_ALOCACAO_FERRAMENTA NUMBER     Identificador da Alocação de uma Ferramenta usada por um Fornecdor para produzir uma determinada Peça. Número sequencial criado automaticamente pelo Sistema.
N_CAPACIDADE_INICIAL_PRODUCAO NUMBER     Capacidade inicial prevista para produção por essa alocação de ferramenta. Valor em Número de Peças.
D_INICIO_USO DATE     Data de Início da Alocação no Fornecedor.
D_FIM_USO DATE Y   Data Final de Uso da Alocação da Ferramenta no Fornecedor.
N_PORCENTAGEM_AVISO NUMBER(2)     Porcentagem de produção da ferramenta a partir da qual os usuários serão avisados que a vida útil da ferramenta está terminando. O Valor padrão, caso não se preencha a informação, será 70%.
ID_PECA NUMBER     Chave estrangeira da tabela PECA e indica qual é a peça produzida pela Ferramenta no Fornecedor.
