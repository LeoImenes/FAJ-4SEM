Table ACRESCIMO_PRODUCAO

Tabela para armazenar as alterações efetuadas na Capacidade de Produção Inicial da alocação de uma Alocação de Ferramenta em um Fornecedor para produzir determinada peça.
Columns
Name Type Optional Default Comments
ID_ACRESCIMO_PRODUCAO NUMBER     Identificador do Acrescimo de Produção. Número sequencial gerado automaticamente pelo Sistema.
ID_ALOCACAO_FERRAMENTA NUMBER     Chave estrangeira da tabela ALOCACAO_FERRAMENTA e identifica para qual Alocação e ferramenta foi feita o incremento na capacidade inicial da ferramenta.
N_ACRESCIMO_PRODUCAO NUMBER     Valor do Incremento de produção, em número de peças, a ser considerado para a alocação da ferramenta.
D_DATA DATE     Data de cadastro do Incremento.
S_MOTIVO VARCHAR2(127) Y   Descrição do Motivo do Incremento.
