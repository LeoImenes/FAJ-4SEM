Table CURSARAM

Tabela para armazenar as disiplinas que já foram cursadas pelos alunos
Columns
Name Type Optional Default Comments
DATA_CONCLUSAO DATE     Data de conclusão da Disciplina.
FREQUENCIA NUMBER     Frequencia que o aluno obteve nas aulas. Deve conter números de 0 a 1, sendo 0 sem freqeucia e 1 100% de frequencia.
NOTA NUMBER     Nota que o aluno obteve ao cursar a Disciplina
SITUACAO VARCHAR2(1)     FLAG para indicar se o usuário foi aprovado ou não na Disciplina. A indica aprovado e R reprovado. Tal campo deverá ser preenchido automaticamente pelo Banco de Dados.
COD_ALUNO NUMBER     Chave estrangeira da tabela ALUNO e indica qual aluno cursou a Disciplina.
COD_DISCIPLINA NUMBER     Chave estrangeira da tabela DISICPLINA e indica qual foi a Disciplina cursada.
