Table ST_DISPOSITIVO

Tabela para armazenar os diversos dispositivos pertencentes à Empresa
Columns
Name Type Optional Default Comments
ID_DISPOSITIVO NUMBER     Chave primária da tabela, formada por um número sequencial.
ID_MODELO_DISPOSITIVO NUMBER     Chave estrangeira da tabela ST_MODELO_DISPOSITIVO e indica qual é o Modelo do Dispositivo
S_NUMERO_SERIE VARCHAR2(20)     Número de Série do Dispositivo. Não existem 2 dispositivos com mesmo Número de Série.
S_ATIVO VARCHAR2(1)   'S' FLAG para indicar se um dispositivo está apto para USO (S) ou não (N). Os não aptos para uso indicam dispositivos quebrados, perdidos ou outras classificações que impedem o USO.
